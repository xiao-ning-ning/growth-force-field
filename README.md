# 成长透镜 / GrowthLens

### AI-Powered Capability Analysis & Behavioral Simulation for Managers

> 知是行之始，行是知之成。先知己之所能，方知己之所向。

**让管理者的软性能力，从"感觉"变成"证据"；让团队的决策风险，从"猜"变成"推演"。**

上传一段会议录音，AI 分析你的行为模式 → 量化能力维度 → 设计可落地的修炼路径 → 创建数字分身推演团队交互。

开源免费 · 数据本地存储 · 无需注册 · 多用户支持

## 解决什么问题

管理者面临的三大困境：

场景 | 痛点 |
|------|------|
述职季 | 下属写了 20 页 PPT，哪些是真正的能力，哪些只是话术？ |
晋升决策 | 凭印象打分，软性能力谁更强？说不清。 |
团队变革 | 推新流程/换负责人，团队会怎么反应？谁抵触、谁先跑？ |

**成长透镜的思路**：软性能力不是靠"问"，而是靠"观察行为"来推断。你做的事、说的话，比你对自己的评价更真实。更进一步——把这些行为模式参数化成数字分身，就能在真实决策前看到团队的可能反应。

## 核心功能

### 1. AI 行为分析
从会议记录、述职自评、访谈纪要中识别真实行为模式，量化已具备、待发展、盲区。每条结论都附**原文引用**，结论可追溯。

**星级评分**：每条证据标注正面（+1星）或负面（-1星），星级累加决定能力归属——正星→已具备，负星→待发展，星级归零→维度移除。双向流动，动态升降。

**录音预处理**：分析前自动做 ASR 纠错、语气词删除、断句修复，显著提升证据引用可读性。

**证据编辑**：可手动修正证据引用文本（保留原始引用可还原），精准控制星级走向。

### 2. 能力图谱
五维全局概览，看到能力全貌和各维度之间的关联。支持展开/折叠、批量删除、维度合并。

### 3. 核心组合
把相关联的碎片能力打包成一个可解释的整体。知道它们是怎么配合的，强在哪里、什么时候用。动态门槛——只有证据数达到分位数才能形成组合，避免弱维度污染。

### 4. 盲区探测
AI 基于行为模式反推"你应该有，但自己没意识到"的能力——不是从录音中直接提取，而是通过跨维度对比、场景覆盖分析识别感知盲区。

### 5. 修炼路径
针对每个待发展维度或盲区，AI 设计具体的修炼步骤——每一步都标注以哪个已有能力作为杠杆。支持勾选步骤完成、标记路径完成。

最小改变，最大效果。不需要推翻重来。

### 6. 成长轨迹
五维能力折线图，追踪能力增长曲线。管理员可对比多个用户的折线，鼠标悬停查看任意日期的详细数据。

### 7. 深度思考
基于已有能力数据，进行认知层面的深度审计。

**深度分析报告**：悖论型框架（默认）与收敛型框架（特例）自动切换。悖论型输出认知锚点→悖论链→不可见结构与风险→行动与叩问；收敛型输出核心论点→证据链→行动方案。所有报告同时输出**易懂版本**（一句话定性 + 具象类比 + 行动要点），并附**报告声明**（具体化本次分析局限）。

**对话模块**：AI 作为专业顾问（有判断力、可反驳、不敷衍），支持多轮对话探讨真实问题，可注入分析上下文。

**摘要笔记本**：实时记录思考，支持 Markdown 渲染与格式工具栏，自动持久化。

### 8. 自定义能力维度（管理员）
上传 Excel 定义自己的维度体系（一级分类、子维度名称、维度定义、已具备/待发展判断标准），AI 严格按你的标准分析，不受通用框架约束。可随时删除恢复 AI 自由生成模式，上传前自动备份旧 schema。

### 9. 数字分身（管理员）
从用户的能力维度数据中，AI 自动提取完整人格画像——一句话定性、人格隐喻、核心倾向与悖论、立场信条与底线、行为反应链（面对问题/方案/情绪/模糊的完整因果链）、动机驱力三层穿透（表层→深层→隐性）、成长方向与突破点。

所有人格参数均可手动编辑微调，确保分身符合你对这个人的真实认知。

### 10. 行为预测模拟（管理员）
选定多个数字分身，设定事件场景和角色配置（职位、汇报关系），AI 进行多轮交互模拟推演。

每轮输出每个分身的内心反应、外部行动、情绪状态、深层驱力，以及系统状态和浮现风险。最终轮输出结局总结、关键发现、管理风险、建议。

**模拟对话**：两种模式——角色对话（与指定分身第一人称对话）和整体讨论（与行为分析专家讨论模拟结果）。

### 11. 团队管理视角（管理员）
管理工具包含三个子标签：
- **团队看板**：热力图纵览团队能力分布，多用户对比折线图，成长记录管理
- **数字分身**：分身列表、创建、查看详情、编辑、删除
- **行为预测**：发起模拟、查看历史、模拟对话

| 对比维度 | 成长透镜 | 传统评估 | 在线 AI 平台 |

## 快速开始

### 前置依赖

| 依赖 | 说明 |
|------|------|
| **Node.js ≥ 18** | 未安装请前往 [nodejs.org](https://nodejs.org) 下载 LTS 版安装；Windows 也可运行 `winget install OpenJS.NodeJS.LTS`；macOS：`brew install node` |
| **AI API Key** | 需要一个 OpenAI 兼容接口的 API Key（推荐 DeepSeek） |
| **.env 配置文件** | 首次启动时服务会自动创建默认配置，只需填入 API Key 即可使用 |

> `start.bat` 会自动检测 `node_modules` 是否存在，若缺失则自动运行 `npm install`。`git clone` 下载包含 `node_modules`，ZIP 下载不包含但会自动补装。

### 启动步骤

**Windows**：
```bash
git clone https://github.com/xiao-ning-ning/growth-lens.git
cd growth-lens
start.bat
```

**macOS / Linux**：
```bash
git clone https://github.com/xiao-ning-ning/growth-lens.git
cd growth-lens
chmod +x start.sh && ./start.sh
```

> 自动检测端口占用、杀掉旧进程、启动服务、打开浏览器。

首次登录默认账号：`admin / admin123456`，请尽快修改密码。

### 配置项

在项目根目录创建/编辑 `.env` 文件：

```env
# AI API（必填）
OPENAI_API_KEY=sk-your-api-key
OPENAI_BASE_URL=https://api.deepseek.com     # 可选，默认 OpenAI 官方
OPENAI_MODEL=deepseek-chat                    # 可选，默认 gpt-4o
OPENAI_TIMEOUT=300000                         # 可选，默认 5 分钟（毫秒）

# 管理员账号（可选，不填则默认 admin / admin123456）
# ADMIN_USERNAME=admin
# ADMIN_PASSWORD=your-password

# 服务配置（可选）
# PORT=3000
# HOST=0.0.0.0
# SESSION_SECRET=auto-generated               # 首次启动自动生成，无需手动设置
```

**模型推荐**：优先使用 **DeepSeek**（性价比高、中文理解好、JSON 输出稳定）。支持 OpenAI / DeepSeek / 月之暗面 / MiniMax / 本地 Ollama（任意 OpenAI 兼容接口均可）。推理模型（DeepSeek-R1）的思考过程会自动剥离。
