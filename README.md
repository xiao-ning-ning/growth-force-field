# 成长透镜 / GrowthLens

### AI-Powered Capability Analysis & Behavioral Simulation for Managers

> 知是行之始，行是知之成。先知己之所能，方知己之所向。

**让管理者的软性能力，从"感觉"变成"证据"；让团队的决策风险，从"猜测"变成"推演"。**

上传一段会议录音，AI 分析你的行为模式 → 量化能力维度 → 设计可落地的修炼路径 → 创建数字分身 → 推演团队交互风险。

开源免费 · 数据本地存储 · 无需注册

## 解决什么问题

管理者面临的三大困境：

场景 | 痛点 |
|------|------|
述职季 | 下属写了 20 页 PPT，哪些是真正的能力，哪些只是话术？ |
晋升决策 | 凭印象打分，软性能力谁更强？说不清。 |
团队决策 | 新流程上线，团队会怎么反应？谁抵触、谁先适应？没人知道。 |

**成长透镜的思路**：软性能力不是靠"问"，而是靠"观察行为"来推断。你做的事、说的话，比你对自己的评价更真实。更进一步——当你把行为模式变成可参数化的数字分身，你就能在真实决策前看到团队的可能反应。

## 核心功能

### 1. AI 行为分析
从会议记录、述职自评、访谈纪要中识别真实行为模式，量化已具备、待发展、盲区。每条结论都附**原文引用**，结论可追溯。

**星级评分**：每条证据标注正面（+1星）或负面（-1星），星级累加决定能力归属——正星→已具备，负星→待发展，星级归零→维度移除。双向流动，动态升降。

**录音预处理**：分析前自动做 ASR 纠错、语气词删除、断句修复，显著提升证据引用可读性。

**证据编辑**：可手动修正证据引用文本（保留原始引用可还原），精准控制星级走向。

### 2. 能力图谱
五维全局概览（战略与诊断、管控与绩效、人心与温度、知识与赋能、制度与设计），看到能力全貌和各维度之间的关联。

### 3. 核心组合
把相关联的碎片能力打包成一个可解释的整体。知道它们是怎么配合的，强在哪里、什么时候用。

### 4. 盲区探测
AI 基于行为模式反推"你应该有，但自己没意识到"的能力——不是从录音中直接提取，而是通过跨维度对比、场景覆盖分析识别感知盲区。

### 5. 修炼路径
针对每个待发展维度或盲区，AI 设计具体的修炼步骤——每一步都标注以哪个已有能力作为杠杆。

最小改变，最大效果。不需要推翻重来。

### 6. 成长轨迹
五维能力折线图，追踪能力增长曲线。管理员可对比多个用户的折线。鼠标悬停查看任意日期的详细数据。

### 7. 深度思考
基于已有能力数据，进行认知层面的深度审计。三栏布局：左侧分析报告、中间对话、右侧摘要笔记本。

**深度分析报告**：
- 悖论型框架（默认）与收敛型框架（特例）自动切换
- 认知锚点 → 悖论链 → 不可见的结构与风险 → 行动与叩问
- 配套"易懂版本"（一句话定性 + 具象类比 + 行动要点）
- 报告声明（具体化本次分析局限，不泛泛而谈）

**对话模块**：AI 作为专业顾问（有判断力、可反驳、不敷衍），支持多轮对话，注入分析上下文。

**摘要笔记本**：支持 Markdown 渲染与编辑（格式工具栏），自动实时持久化。

### 8. 数字分身（管理员）
从用户的能力维度数据中，AI 自动提取完整人格画像——包括身份背景、人格隐喻、核心倾向、悖论、盲区、立场信条、行为反应链、动机驱力三层穿透、成长方向。所有参数均可手动编辑微调。

### 9. 行为预测模拟（管理员）
选定多个数字分身，设定事件场景和角色配置（含汇报关系），多轮交互模拟推演。

- 每轮输出每个分身的内心反应、外部行动、情绪、驱力
- 系统状态与浮现风险
- 最终轮输出结局总结、关键发现、管理风险、建议
- 模拟对话：可与分身角色第一人称对话，或与行为分析专家讨论结果

### 10. 自定义能力维度（管理员）
上传 Excel 定义自己的维度体系（一级分类、子维度名称、维度定义、已具备/待发展判断标准），AI 严格按你的标准分析，不受通用框架约束。可随时删除恢复 AI 自由生成模式。

### 11. 团队管理视角（管理员）
管理工具包含三个子标签：
- **团队看板**：热力图纵览团队能力分布，多用户对比折线图
- **数字分身**：创建、查看详情、编辑、删除分身
- **行为预测**：发起模拟、查看模拟历史、模拟对话

## 差异化

| 对比维度 | 成长透镜 | 传统评估 | 在线 AI 平台 |
|---|---|---|---|
| 证据链 | **每条结论附原文引用** | 无 | 无 |
| 数据存储 | **本地**，不出公司 | 云端或纸质 | 云端 |
| 持续积累 | **增量生长**，不是每次重新分析 | 一次性 | 一次性 |

| 行为模拟 | **数字分身+多智能体推演** | 无 | 无 |
| 认知深度 | **悖论型框架+易懂版本** | 通用评分 | 通用评分 |

## 快速开始

### 前置依赖

| 依赖 | 说明 |
|------|------|
| **Node.js** | 未安装请前往 [nodejs.org](https://nodejs.org) 下载 LTS 版安装；Windows 也可运行 `winget install OpenJS.NodeJS.LTS`；macOS：`brew install node` |
| **.env 配置文件** | 首次启动时服务会自动创建默认配置，无需手动创建 |

> `start.bat` 会自动检测 `node_modules` 是否存在，若缺失则自动运行 `npm install`。`git clone` 下载包含 `node_modules`，ZIP 下载不包含但会自动补装。

### 启动步骤

**Windows**：
```bash
git clone https://github.com/xiao-ning-ning/growth-lens.git
cd growth-lens
start.bat
```

**macOS / Linux**：
```bash
git clone https://github.com/xiao-ning-ning/growth-lens.git
cd growth-lens
chmod +x start.sh && ./start.sh
```

> 自动检测端口占用、杀掉旧进程、启动服务、打开浏览器。

### 配置项（可选）

在项目根目录创建/编辑 `.env` 文件：

```env
OPENAI_API_KEY=sk-your-api-key        # 必填，AI API 密钥
OPENAI_BASE_URL=https://api.deepseek.com  # 可选，默认 OpenAI 官方
OPENAI_MODEL=deepseek-chat            # 可选，默认 gpt-4o
OPENAI_TIMEOUT=300000                 # 可选，默认 5 分钟（毫秒）
ADMIN_USERNAME=admin                   # 可选，默认 admin
ADMIN_PASSWORD=your-password           # 可选，默认 admin123456
SESSION_SECRET=                        # 可选，首次启动自动生成
PORT=3000                              # 可选，默认 3000
HOST=0.0.0.0                           # 可选，默认 0.0.0.0
```

**模型推荐**：优先使用 **DeepSeek**（性价比高、中文理解好、JSON 输出稳定）。支持 OpenAI / DeepSeek / 月之暗面 / 本地 Ollama（任意 OpenAI 兼容接口均可）。推理模型（DeepSeek-R1、MiniMax 等）的思考过程会自动剥离。
